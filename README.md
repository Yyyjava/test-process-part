### I. 小组使用的工具与具体合作方式

#### 📅 会议 [![Notion](https://img.shields.io/badge/-Notion-000000?style=flat&logo=notion&logoColor=white)](https://notion.so)
* **形式**：线下利用课后时间面对面讨论。
* **记录**：所有会议纪要均同步至 Notion 任务看板。

#### 🌿 版本控制 [![GitHub](https://img.shields.io/badge/-GitHub-181717?style=flat&logo=github&logoColor=white)](https://github.com)
* 所有代码提交和版本发布的**唯一平台**。
* 遵循 Git Flow 工作流，确保主分支稳定性。
